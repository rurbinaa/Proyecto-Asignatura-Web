# Estructura de Archivos Excel para Importación

Este documento describe la estructura exacta que deben tener los archivos Excel para ser compatibles con el sistema de importación de datos de calidad. El sistema procesa 5 hojas específicas de un archivo Excel (.xlsx), cada una con columnas específicas que se mapean a campos de la base de datos.

## Notas Importantes
- Los archivos deben estar en formato .xlsx (Excel 2007+).
- Las columnas deben estar en el orden especificado, aunque el sistema mapea por nombre.
- Los tipos de datos indicados son los esperados; el sistema realiza conversiones automáticas donde es posible.
- Columnas faltantes se rellenan con valores por defecto (0 para numéricos, "UNKNOWN" para texto).
- Los defectos se almacenan en tablas separadas con relaciones Many-to-Many.</content>
